## 🚀 실행 방법
- 아래 방법을 따르되 각자의 환경에 맞추어 Python, Jupyter Notebook을 설치하여 진행해 주세요. 

1️⃣ **저장소 클론**
```bash
git clone https://github.com/<your-username>/<repo-name>.git
```

2️⃣ **가상환경 생성 및 Jupyter 설치**
```bash
pip install jupyter
```

3️⃣ **Jupyter Notebook 실행**
```bash
jupyter notebook
```

4️⃣ 원하는 장(`.ipynb` 파일)을 열어 학습을 시작하세요.
